WEB-приложение представляет собой сайт IT-комапании.
Приложение создано на основе фреймворка ASP.NET Core MVC с использованием Entity Framework. Также используется фреймворк Bootstrap.

На сайте можно оставлять заявки, которые в дальнейшем будут обрабатываться администратором.
Администратор может присваивать заявкам различный статус в зависимости от их текущего состояния,
а также фильтровать их по различным параметрам.
Кроме того, на сайте имеется информация о проектах выполненных компанией и услугах предоставляемых компанией (администратор может обновлять эти разделы).
Присутствует также блог, куда администратор может публиковать статьи.
